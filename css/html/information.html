HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a web page.

HTML consists of elements, which are enclosed by tags. Each element can have attributes that define its behavior or appearance.

Here's an example of a basic HTML structure:

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <title>My Website</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
            </body>
            </html>
